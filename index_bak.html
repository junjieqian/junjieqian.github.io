Skill Set

  • coding

    Writing code is a natural component of my regular workflow. I have experience building software with system-level languages like C/C++ and Java, as well as employing scripting languages like Python and Bash to process and clean raw data files. A full list of familiar languages and software tools can be found on my resume.

  • visualization

    Nearly all of my projects have to deal with large or huge datasets (hundreds GBs), and so visualize these data is key to do the analyzation. In my own experience, I build most of the tools by myself using Python and Matplotlib, besides I also use GNU Octave/Matlab and R for iterative exploration while developing algorithms.

  • modeling

    Most of my projects involve performance optimization, and so it is important to build the model with cost and gain for the application or system. I use various profiling tools to measure the performance metrics, including CPU utilization, memory usage, I/O throughput/latency and so on.

  • systems

    Familiarity with operating system and storage system design has yielded many benefits for me on performance optimization. First, I have better understanding of possible performance bottlenecks and the quick ways to locate them. Second, I have exploited the interaction of parallelism in runtime systems (e.g. JVM) and underlying OS.

Projects

  • Color Thief

    Color Thief is an iOS app for color transfer. It harnesses a fast GPU code I wrote for quickly swapping colors between photos using phone hardware.

  • Overview

    Overview is a tool for exploring large, unlabelled document datasets. It is targeted at helping journalists quickly analyze text dumps.

  • Glimmer

    Glimmer is an algorithm for fast multidimensional scaling. Significant speed gains are achieved by leveraging GPU parallelism.

  • I1 Optimization

    I1 is an algorithm for stochastic optimization based on Information Filtering. It is designed to complement the optimization algorithm called K1 based on the Kalman Filter.

  • Minimum Degree Matrix Reordering Tutorial

    I wrote a tutorial on different matrix reordering algorithms. Minimum degree matrix reordering increases the efficiency of the matrix inverse.

  • Small World Graph Visualization

    I built a graph visualization tool based on Jarke van Wijk's paper using the prefuse toolkit.

  • MetaCombine

    The MetaCombine Project at Emory University assessed using semantic clustering as a means of exploring digital libraries.

  • TeamView

    TeamView is a tool for analyzing and labelling biological tracking data. It was used by researchers in the BORG Lab at Georgia Tech.

  • Nonnegative Matrix Factorization

    This report describes an algorithm for computing fast nonnegative matrix factorizations.

  • Growbot

    In 2004, my PC game, Growbot, was one of the Student Showcase winners in the Independent Games Festival.
